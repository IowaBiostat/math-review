2.6 Logarithm and exponential

Exponential definition

The exponential function is f(x)=axf(x) = a^xf(x)=ax; the base aaa must be a positive real number but the exponent xxx can be any real number. The precise definition doesn’t come up often, but here it is in case you ever need it (defining all these cases is necessary in order to ensure that the resulting function is continuous:

  1. If xxx is a positive integer nnn, then an=a⋅a⋯a(na^n = a \cdot a \cdots a \quad (nan=a⋅a⋯a(n times)
  2. If x=0x=0x=0, then a0=1a^0 = 1a0=1
  3. If xxx is a negative integer, then a−n=1ana^{-n} = \frac{1}{a^n}a−n=an1​
  4. If xxx is a rational number p/qp/qp/q, with q>0q>0q>0, then ap/q=apqa^{p/q} = \sqrt[q]{a^p}ap/q=qap​
  5. If xxx is an irrational number, then it’s defined as the limit of ara^rar, where rrr is a sequence of rational numbers whose limit is xxx.

Note that we would run into trouble at step 4 if we tried to allow negative bases.

Logarithm definition

The logarithmic function with base aaa is defined as the function satisfying

log⁡ax=y  ⟺  ay=x\log_a x = y \iff a^y = xloga​x=y⟺ay=x

If we leave off the base, it is assumed to be base eee, the “natural logarithm”:

log⁡x=log⁡ex\log x = \log_e xlogx=loge​x

log⁡x=y  ⟺  ey=x;\log x = y \iff e^y = x;logx=y⟺ey=x;
