Exercise: Write an R function n(eps) that returns the smallest NNN for which n>N  ⟹  ∣f(xn)−f(x0)∣<ϵn > N \implies \lvert f(x_n)-f(x_0)\rvert < \epsilonn>N⟹∣f(xn​)−f(x0​)∣<ϵ for xn=21/nx_n = 2^{1/n}xn​=21/n and f(x)=exf(x) = e^xf(x)=ex.

Conceptually, this is a three-part process:

  1. Determine what xnx_nxn​ is converging to. Here, xn→1x_n \to 1xn​→1.
  2. Determine the largest value of delta that satisfies e1+δ−e1<ϵe^{1+\delta} - e^1 < \epsilone1+δ−e1<ϵ.
  3. Determine the smallest value of NNN such that 21/n−1<δ2^{1/n} - 1 < \delta21/n−1<δ.
n <- function(eps) {
  delta <- log(eps + exp(1)) - 1
  ceiling(1/log2(1+delta))
}
