3.1 Definitions and conventions

A matrix is a collection of numbers arranged in a rectangular array of rows and columns, such as

[324−1−12]\left[ \begin{array}{rr} 3 & 2 \\ 4 & -1 \\ -1 & 2 \end{array} \right]​34−1​2−12​​

A matrix with rrr rows and ccc columns is said to be an r×cr \times cr×c matrix (e.g., the matrix above is a 3×23 \times 23×2 matrix).

In the case where a matrix has just a single row or column, it is said to be a vector, such as

[3−1]\left[ \begin{array}{r} 3 \\ -1 \end{array} \right][3−1​]

Conventionally, vectors and matrices are denoted in lower- and upper-case boldface, respectively (e.g., xxx is a scalar, x\mathbf{x}x is a vector, and X\mathbf{X}X is a matrix). In addition, vectors are taken to be column vectors – i.e., a vector of nnn numbers is an n×1n \times 1n×1 matrix, not a 1×n1 \times n1×n matrix.

The ijijijth element of a matrix M\mathbf{M}M is denoted by MijM_{ij}Mij​ or (M)ij(\mathbf{M})_{ij}(M)ij​.

For example, letting M\mathbf{M}M denote the above matrix, M11=3M_{11}=3M11​=3, (M)32=2(\mathbf{M})_{32}=2(M)32​=2, and so on. Similarly, the jjjth element of a vector v\mathbf{v}v is denoted vjv_jvj​; e.g., letting v\mathbf{v}v denote the above vector, v1=3v_1 = 3v1​=3.
